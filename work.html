How it works
4-Agent Orchestration Pipeline

Each agent is a specialist. A central orchestrator feeds their outputs into the next stage, forming a long-chain reasoning pipeline.

📋
Planner
Problem Decomposition
Breaks your complex question into 3-5 targeted subtopics for deep investigation.
🔍
Searcher
Parallel Web Research
Executes parallel web searches across all subtopics, gathering raw evidence and sources.
📝
Summarizer
Structured Analysis
Extracts key findings, data points, and insights from raw search results.
📄
Writer
Report Generation
Synthesizes all analyses into a coherent, professional research report.

Capabilities
Built for Real Research Work
🧩
Long-Chain Reasoning
Multi-step planning that decomposes complex problems into tractable sub-problems before execution begins.
⚡
Parallel Execution
Subtopic searches run concurrently, cutting total research time dramatically compared to sequential approaches.
🛡️
Fault Tolerance
Each agent has retry logic and graceful degradation. If one search fails, the pipeline continues with available data.
🔌
Pluggable Architecture
Swap any agent or tool independently. Works with OpenAI, DeepSeek, Qwen, or any OpenAI-compatible endpoint.
📊
Structured Output
Reports include key findings, data points, source citations, confidence ratings, and identified information gaps.
🌐
Live Web Access
Agents search the real web in real-time — no static knowledge cutoff. Results are fresh and current.
